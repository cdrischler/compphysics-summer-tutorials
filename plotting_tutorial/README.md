## Troubleshooting

If the menu bar in `xmgrace` is scrambled in your virtual machine, change the setting in VirtualBox's menu bar as follows:
* `View > Virtual Screen 1 > Resize to 1600x900`
* `View > Virtual Screen 1 > Scale to 100%`
